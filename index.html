React

Biblioteca de JavaScript creada por Facebook para construir interfaces de usuario dinámicas y reutilizables mediante componentes.

PHP

Lenguaje del lado del servidor, popular para aplicaciones web dinámicas y bases de datos.

Mongo DB

Base de datos NoSQL orientada a documentos que almacena datos en formato JSON flexible, ideal para aplicaciones escalables y modernas.

MySQL

Sistema de gestión de bases de datos relacional (RDBMS) de código abierto, ideal para almacenar datos estructurados y realizar consultas eficientes con SQL.

CSS

Lenguaje de estilo utilizado para diseñar y dar formato a las páginas web, controlando la apariencia y el diseño visual de los elementos HTML.

SaSS

Preprocesador de CSS que añade características como variables, anidamiento y funciones, facilitando la escritura y mantenimiento de estilos más organizados.

Express.js

Framework minimalista y flexible para Node.js que facilita la creación de servidores y APIs web de manera rápida y eficiente.

Sequelize.js

ORM (Object-Relational Mapping) para Node.js que permite interactuar con bases de datos SQL como MySQL, PostgreSQL y SQLite usando JavaScript.

TailwindCSS

Framework CSS basado en utilidades, que permite crear diseños personalizados y responsivos rápidamente mediante clases predefinidas.

BootStrap

Framework de CSS y JavaScript que facilita el diseño web responsivo y la creación rápida de interfaces con componentes predefinidos y personalizables.

HTML5

La última versión del lenguaje HTML, que estructura y presenta contenido web, con soporte mejorado para multimedia y elementos interactivos.
